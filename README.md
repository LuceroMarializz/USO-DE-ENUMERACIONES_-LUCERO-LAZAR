# Mi definicion de Enum

"El uso del enum nos ayuda a poder darle nombre a un tipo de dato y convertirlo a un conjunto de variables legibles por defecto."

# Explicación de mi definicion:

| Parte de la definición | Significado |
|------------------------|-------------|
| "darle nombre a un tipo de dato" | Creamos un tipo de dato personalizado (ej: `EstadoSolicitud`) |
| "convertirlo a un conjunto de constantes" | Ese tipo de dato contiene valores fijos (ej: `Pendiente`, `EnProceso`) |
| "legibles por defecto" | Los valores se muestran como palabras, no como números |

#Ejemplo visual:

// SIN enum - números poco legibles
int estado = 1;  // ¿Que significa 1?

// CON enum - nombres legibles por defecto
EstadoSolicitud estado = EstadoSolicitud.Pendiente;  // Se entiende solo
